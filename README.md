# Spring Security Demo

A Spring Boot application demonstrating Spring Security authentication and authorization.

## Features

* Spring Security integration
* User authentication
* Role-based authorization
* Protected REST endpoints
* Password encryption

## Technologies

* Java
* Spring Boot
* Spring Security
* Maven

## Getting Started

### Clone the repository

```bash
git clone https://github.com/ConnectRanju/spring-security-demo.git
```

### Run the application

```bash
mvn spring-boot:run
```

## Project Structure

* `config` - Security configuration
* `controller` - REST controllers
* `service` - Business logic
* `repository` - Data access layer
* `model` - Domain models
